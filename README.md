# Loan-Approval-Prediction

## Setting up the Environment

To set up the virtual environment and install the required dependencies, follow these steps:

1. **Create a virtual environment:**
   ```bash
   python3 -m venv venv
   ```

2. **Activate the virtual environment:**
   - On Linux/MacOS:
     ```bash
     source venv/bin/activate
     ```

3. **Install the required packages:**
   ```bash
   pip install -r requirements.txt
   ```

## Running the Notebook

To run the Jupyter Notebook, follow these steps:

1. **Ensure the virtual environment is activated:**
   ```bash
   source venv/bin/activate
   ```

2. **Install Jupyter if not already installed:**
   ```bash
   pip install jupyter
   ```

3. **Launch the Jupyter Notebook:**
   ```bash
   jupyter notebook loan_approval_prediction.ipynb
   ```

4. **Run the cells in the notebook:**
   - Open the notebook in the browser.
   - Execute the cells sequentially to ensure proper execution of the code.
